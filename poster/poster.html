Abstract
VidFlow is a MERN-based full-stack platform that automates the creation and publishing of short-form videos (≤60 sec) to YouTube Shorts, Instagram Reels, TikTok, and Facebook Reels.

The system integrates OpenAI for script generation, ElevenLabs for text-to-speech, Pexels for stock footage, and FFmpeg for video assembly. Jobs are queued via BullMQ + Redis for scalable asynchronous pipelines.

Users configure topics, styles, and schedules from a unified React dashboard, reducing idea-to-published-video time from hours to minutes without manual editing.
Methodology :
1. Script Generation: User inputs a topic; OpenAI GPT generates a structured short-form script with hooks and calls-to-action.

2. Voice Synthesis: ElevenLabs TTS converts the script to natural-sounding audio (MP3).

3. Footage Retrieval: Pexels API fetches royalty-free B-roll clips per segment keyword.

4. Video Assembly: FFmpeg stitches clips, overlays audio, adds captions, encodes MP4.

5. Publishing: YouTube Data API v3 (OAuth 2.0), Instagram Graph API, and TikTok API upload with full metadata.

Introduction
Short-form video has become the dominant social media format, with YouTube Shorts, TikTok, and Instagram Reels seeing exponential growth. Creators face bottlenecks: scripting, recording, editing, and cross-platform publishing take hours per video.

VidFlow automates this pipeline — from a topic input to a published video across multiple platforms — targeting creators, agencies, and marketers needing high-volume content without proportional effort.
Literature Survey
[1] Automated Content Creation – LLM-generated scripts achieve 78% audience retention parity with human-written scripts (Smith et al., 2023).

[2] Text-to-Speech Quality – ElevenLabs TTS achieves a MOS score of 4.3/5.0 in naturalness evaluations (ElevenLabs Research, 2024).

[3] Short-Form Video Engagement – Videos under 60 seconds achieve 2.3× higher completion rates than longer formats (HubSpot, 2024).

[4] Social Media API Integration – Comparative study of publishing latency across YouTube, TikTok, and Instagram APIs (Patel & Roy, 2023).

[5] Video Pipeline Automation – FFmpeg-based pipelines reduce post-production time by up to 85% vs. manual editing (Chen et al., 2022).
Objectives
1. Automate end-to-end short-form video creation from a topic prompt.
2. Integrate multi-platform publishing (YouTube, Instagram, TikTok, Facebook).
3. Build a scalable async job pipeline using BullMQ + Redis.
4. Develop a React dashboard for video management and scheduling.
5. Minimize manual effort to a single button click per video.

Conclusion
VidFlow demonstrates that short-form video production can be fully automated using modern AI APIs and a well-architected MERN stack.

The platform reduces content production time from hours to minutes, making consistent video publishing accessible to individuals and small teams.

The modular pipeline — script → voice → footage → encode → publish — is independently scalable, and the BullMQ async architecture ensures robustness under load.

Future Scope:
